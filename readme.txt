How to run the derraw_adoption_agency website
*************************************************
1. Download the  zip file

2. Extract the file and copy derraw_adoption_agency folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name 'derraw_adoption_agency'.

6. Import 'derraw_adoption_agency.sql' file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/derraw_adoption_agency/ (frontend)
